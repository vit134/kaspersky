# Тестовое задание

> A Vue.js + node.js + redis project

## Build Setup

``` bash
# install dependencies
npm install
cd server npm i

# run client with hot reload
npm run dev

# run server
npm run server

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run all tests
npm test

```
